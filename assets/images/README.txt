MUSE BY ANAHITA - CUSTOM ASSETS DIRECTORY
=========================================

This directory is dedicated to your local, custom salon images. 

Vite automatically serves any files placed inside the "/public" directory at the root level.
This means you can place your images here and easily reference them in the application!

Recommended Image Map for Customizing Your Website:
---------------------------------------------------

1. CATALOGUE CAROUSEL IMAGES
   To replace the 3D Carousel images, place your images in this folder and name them:
   - bridal.jpg       (Recommended size: 800x1200px)
   - hair.jpg         (Recommended size: 800x1200px)
   - makeup.jpg       (Recommended size: 800x1200px)
   - skincare.jpg     (Recommended size: 800x1200px)
   - nails.jpg        (Recommended size: 800x1200px)
   - spa.jpg          (Recommended size: 800x1200px)
   - henna.jpg        (Recommended size: 800x1200px)
   - styling.jpg      (Recommended size: 800x1200px)

   Then, you can update the src/components/CatalogueCarousel.tsx image paths to:
   - "/assets/images/bridal.jpg"
   - "/assets/images/hair.jpg"
   - "/assets/images/makeup.jpg"
   - "/assets/images/skincare.jpg"
   - "/assets/images/nails.jpg"
   - "/assets/images/spa.jpg"
   - "/assets/images/henna.jpg"
   - "/assets/images/styling.jpg"

2. BRIDAL PAGE HERO & SERVICES
   Place your images here and name them:
   - bridal_hero.jpg
   - skincare_treatment.jpg
   - massage.jpg

HOW TO UPLOAD:
--------------
Simply upload your desired image files directly into this directory (/public/assets/images/) using your file explorer or workspace interface.
Once uploaded, your website will immediately render them from the local server.
